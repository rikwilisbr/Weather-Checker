

# Weather Checker Project

"Weather Checker" is a Web Application designed to check the weather details of any place in the world.







![App Screenshot](https://i.imgur.com/jF0srMR.png)

You can access the app here: https://weathercheckersite.herokuapp.com/







## Languages and Tools

- React
- Bootstrap
- OpenWeather API
